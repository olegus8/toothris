бонус активируется после выполнения всех условий и падения текущей фигуры


d супер бросок
    фигура падает на самое дно, разрушая фигуры в стакане на своем пути
    дается за бросок фигуры каждого типа по одному разу

h супер удар
    за каждый удар о борт, разрушается нижняя строка в стакане
    дается за удар фигуры каждого типа по одному разу

b строитель
    после падения фигуры, в стакане застраиваются все пустые ячейки под ее вертикальной проекцией
    дается за приземление фигуры каждого типа по одному разу

k царь горы
    после падения фигуры, уничтожаются все ячейки в стакане на одном уровне с фигурой
    дается за поворот фигуры каждого типа по одному разу

a кислотный дождь
    уничтожаются верхние ячейки каждого столбца в стакане
    дается за разбиение фигуры каждого типа по одному разу

g град
    с неба падают ячейки, выравнивая высоту каждого столбца в стакане
    дается за поглощение фигуры каждого типа по одному разу

q землетрясение
    ячейки в стакане оседают, заполняя свободные пространства
    дается за зависание фигуры каждого типа по одному разу
